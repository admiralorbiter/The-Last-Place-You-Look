# Personas

## Primary persona: Digital Archivist with Messy External Drives

### Summary
A technically capable or semi-technical user with years of files scattered across external hard drives, old folders, project exports, media dumps, and ad hoc backups.

### Environment
- multiple mounted local volumes
- removable external drives
- inconsistent folder structures
- accidental duplicates mixed with intentional mirrors
- media-heavy and document-heavy archives

### Goals
- know what exists
- know what is duplicated versus intentionally backed up
- find the best copy quickly
- reorganize without breaking trust
- avoid deleting the wrong thing

### Frustrations
- not knowing which drive has the real copy
- duplicate sprawl
- inconsistent naming
- fear of cleanup
- having to manually compare files before acting

### What wins them
- a clear first scan summary
- explanations for recommendations
- visible relationship maps
- reversible actions
- protection warnings that feel honest

---

## Secondary persona: Active Project Power User

### Summary
A user who also has live project workflows, but is not the initial design center.

### Environment
- active working folders plus archives
- source files, exports, drafts, and deliverables
- desire to group by project and revisit later

### Goals
- keep active work understandable
- avoid duplicate drift between source and exports
- track which copy is canonical

### Frustrations
- scattered assets across volumes
- project files split from outputs
- hard to reconstruct what belongs together

### What wins them
- project clustering later
- relationship maps for item groups
- smart grouping and lightweight organization

---

## Not primary in MVP
The product is not initially optimized for:
- team collaboration
- enterprise IT asset governance
- cloud-only storage users
- NAS-first households
- users who only want a simple duplicate deleter

## Persona ordering for roadmap
1. messy archive users on mounted local storage
2. users balancing archive + active project workflows
3. broader personal storage users
